================================================================================
                    POLICYBRIDGE AI - COMPLETE PROJECT OVERVIEW
================================================================================

🎯 PROJECT PURPOSE:
PolicyBridge AI is an intelligent insurance policy analysis platform that uses 
Google Gemini AI to automatically compare, analyse, and provide insights on 
insurance policies. It helps insurance professionals make data-driven decisions 
by converting complex policy documents into clear, comparable insights.

🏗️ TECHNICAL ARCHITECTURE OVERVIEW:
- Frontend: React.js (Modern Single Page Application)
- Backend: Django REST Framework (Python-based API server)
- Database: SQLite (Development) / PostgreSQL (Production-ready)
- AI Integration: Google Gemini API for intelligent analysis
- Authentication: JWT (JSON Web Tokens) for secure user sessions
- File Processing: PDF/DOCX text extraction and analysis
